# PBL Question Lab

הופכים נושא לימודי לשאלת PBL שאי אפשר לפתור עם תשובה מגוגל.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Heebo font (Hebrew RTL)
- Lucide React icons
- Anthropic Claude API (`@anthropic-ai/sdk`)
